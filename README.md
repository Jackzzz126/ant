### dependence
* [libjson.a](http://sourceforge.net/projects/libjson/)
* [protobuf2.6.1](https://github.com/google/protobuf/releases/download/v2.6.1/protobuf-2.6.1.zip)

### build
* install make, g++
* run `make` in root dir.
* run `make clean` to clean all tmp file
